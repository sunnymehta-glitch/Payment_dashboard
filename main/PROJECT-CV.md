## CV — Compact (Single Bullet)

**PayBroadcast — Automated Payment Alert System** | *React, Node.js, Twilio, Google Sheets API, Vercel*

- Built a serverless webhook system that listens to Zoho Books invoice events, sends real-time WhatsApp alerts via Twilio API, logs transactions to Google Sheets, and displays live financial data on a React dashboard — eliminating manual payment tracking for the business.

---

## CV — Detailed (Multi-Bullet)

**PayBroadcast — Automated Payment Alert System**
*React.js | Node.js | Express | Twilio API | Google Sheets API | Zoho Books | Vercel Serverless*

- Developed a webhook-driven backend that ingests Zoho Books invoice events, normalizes multi-format payloads, and routes alerts to stakeholders via Twilio WhatsApp API.
- Integrated Google Sheets API with JWT-based service account auth for automated transaction logging, with CSV fallback for resilience.
- Built a real-time React dashboard with financial summaries, payment history, dark/light theme, and auto-refresh polling.
- Implemented a sheet monitoring service that detects manually added entries and triggers WhatsApp notifications automatically.
- Deployed as serverless functions on Vercel for zero-downtime, scalable operation.

### Key Features

- Automated webhook ingestion from Zoho Books with multi-format payload parsing
- Instant WhatsApp alerts to business stakeholders via Twilio
- Google Sheets integration for real-time transaction logging with CSV fallback
- Sheet monitoring service that detects manually added entries and triggers notifications
- Interactive React dashboard with financial overview, payment list, dark/light theme, and auto-refresh
- Serverless architecture deployed on Vercel for scalability and zero maintenance

### Motive / Purpose

To automate the manual process of tracking invoices and notifying team members about new payments. The system eliminates the need to constantly check Zoho Books by pushing instant WhatsApp alerts and maintaining a centralized, always-up-to-date payment log — improving response time, reducing human error, and giving stakeholders real-time financial visibility.
